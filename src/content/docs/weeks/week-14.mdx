## 자주 발생하는 문제와 해결책

| 문제 | 원인 | 해결 |
|------|------|------|
| 에이전트 무한 루프 | 탈출 조건 미설정 | 최대 반복 횟수 hard limit 추가 |
| 컨텍스트 오염 | Context Rot 진행 | 즉시 wipe + 상태 파일 복구 |
| 에이전트 간 충돌 | 동시 파일 수정 | 파일 잠금(Lock) 메커니즘 |
| 비용 폭발 | 무한 API 호출 | 토큰 예산 설정 |
