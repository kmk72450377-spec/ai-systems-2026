## Theory

### Traditional SDLC → Agentic SDLC

| Traditional Role | Agentic Equivalent | Tool Access (MCP) |
|------------------|-------------------|-------------------|
| Product Manager | Planner Agent | Web search, document reading |
| Software Architect | Initializer Agent | Repository mapping, Git branching |
| Developer | Coder Agent (Ralph Loop) | File editing, compiler |
| QA Engineer | QA Agent | pytest, diff viewer |
| DevOps | Deploy Agent | Docker, Kubernetes |

<Aside type="caution" title="Golden Rule of Inter-Agent Communication">
Communication between agents must always occur through **structured artifacts** (Markdown, JSON).
Having Agent A send direct messages to Agent B maximizes non-determinism.
</Aside>

### Artifact-Based Handoff Design

```json
// task_queue.json — Initializer → Coder handoff
{
  "project": "calculator-app",
  "tasks": [
    {
      "id": "task-001",
      "description": "Implement add() function",
      "target_file": "calculator.py",
      "test_file": "tests/test_calculator.py",
      "acceptance_criteria": ["All pytest tests pass", "Type hints included"],
      "status": "pending"
    }
  ]
}
```
